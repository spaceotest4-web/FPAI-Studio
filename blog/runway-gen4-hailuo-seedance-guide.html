Three New Powerhouses

FP AI Studio now features 10 AI video models, and three of the newest additions bring unique capabilities that weren't available before. Runway Gen4 Turbo, Hailuo 02, and Seedance Pro each excel in different scenarios. Here's everything you need to know about them.

Runway Gen4 Turbo

Runway has been a pioneer in AI video generation, and Gen4 Turbo is their latest and fastest model. It's designed specifically for image-to-video generation — you provide a reference image and a prompt, and the model animates the scene.

Key Features

  • Image-to-video focused — Requires an input image URL to animate
  • Multiple aspect ratios — 1280x720, 720x1280, 1104x832, 832x1104, 960x960, and ultra-wide 1584x672
  • Duration options — 5 or 10 second clips
  • Seed support — Reproducible results for iteration
  • 1080p output — High-resolution video output

Best Use Cases

  • Animating product photos for e-commerce
  • Bringing still artwork or illustrations to life
  • Creating subtle motion effects (parallax, camera moves) from photographs
  • Ultra-wide cinematic shots (1584x672 ratio)

Pro Tip

Generate a high-quality image first using Flux 2 Pro or Mystic in FP AI Studio's image generation tab, then feed that image URL into Runway Gen4 to animate it. This two-step workflow gives you maximum control over both the visual quality and the motion.

Hailuo 02 (1080p)

Hailuo is a model that stands out for its prompt optimizer — an AI that rewrites your prompt to produce better results. It's designed to be beginner-friendly while still producing professional-quality output.

Key Features

  • Prompt Optimizer — Toggle this on and the AI rewrites your prompt for optimal results. Great for beginners who aren't sure how to phrase things
  • First & Last Frame — Provide reference images for the opening and closing frames for precise control
  • Fixed 6-second duration — Optimized for a single duration for consistent quality
  • 1080p output — Full HD resolution

Best Use Cases

  • Quick social media clips (6 seconds is perfect for Reels/TikTok hooks)
  • Beginners who want good results without mastering prompt engineering
  • Transition videos (first frame to last frame interpolation)
  • Fast iteration when you need reliable quality

Pro Tip

Use the first and last frame feature to create smooth transitions. Generate two different images (e.g., summer scene and winter scene) and let Hailuo create a seamless 6-second transition between them.

Seedance Pro (1080p)

Seedance Pro is a versatile model that supports both text-to-video and image-to-video generation with a strong feature set including negative prompts and seed control.

Key Features

  • Dual mode — Works as text-to-video (prompt only) or image-to-video (with reference image)
  • Negative prompt support — Exclude unwanted elements from your video
  • Seed control — Reproduce and iterate on specific results
  • Duration options — 5 or 10 second clips
  • 1080p output — Full HD quality

Best Use Cases

  • General-purpose video generation where you want full control
  • Videos where you need to exclude specific elements (negative prompt)
  • Iterative creation where seed reproducibility matters
  • Both text-based concepts and image-based animations

Pro Tip

Use the negative prompt to avoid common AI video artifacts: "blurry, distorted faces, flickering, inconsistent lighting, morphing objects". This significantly improves output quality.

When to Use Each Model

  • Runway Gen4 → You have a specific image you want to animate, especially with unique aspect ratios
  • Hailuo 02 → Quick social clips, beginners, or when you want AI to optimize your prompt
  • Seedance Pro → General-purpose video with full control (negative prompts + seed + optional image)

All three models are available in FP AI Studio's Video Generation tab. Select the model from the dropdown, adjust settings, and generate. Try the same prompt across all three to see which produces the look you prefer.
